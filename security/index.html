At kwatafana.org cyber security is one of our core pillars, we go the extra mile to keep our systems secure.

Security

Security deals with defending against unwanted data controls. Or more precisely enforcing Confidentiality, Integrity and data Availability within a system (notice that these make up the CIA acronym üòú).

We are working on cyrtophora, a fullstack framework for secure internet services. It is a core component in our infrastructure, we use it to write most of our internet services.

Alongside cyrtophora we are writing a cyber-security guide secure bits, it details the defensive techniques used in cyrtophora. The book is meant to be a general guide to cyber-security.

High Assurance Systems

Our software and networking services should be dependable and robust, they should also be highly available and be able to function in safety critical environment. You can read more about this here

Pen-Testing

One way to learn about avenues of attack, is by trying to attack existing systems. We do this with our tool h4, which is meant to test the security of our services but it is generic enough to be used by any cyber security researcher.

We also help organization with their cyber-security, we can try to hack your software or internet services from android apps, web servers to cryptography related systems. We will give you a detailed report of the vulnerabilities and how to prevent them. If you are interested in this please drop us an email kaindume@kwatafana.org
