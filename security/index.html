Defense

Defense deals with defending against unwanted data controls or more precisely enforcing Confidentiality, Integrity and data Availability within the system (notice that these make up the CIA acronym).

At kwatafana.org we mainly study and implement cyber defensive measures through our open source framework cyrtophora, which we also use for writing our services. Cyrtophora also comes with a book that details the defensive techniques used, this book is meant to be a general guide to cyber security even if you are not using cyrtophora.

Offense

Offense deals with gaining unwanted/unauthorized access to data. Essentially this is subverting the Confidentiality, Integrity and data Availability of a system.

At kwatafana.org we mainly study cyber attacks with our open source tool h4, which is meant to test the security of kwatafana.org services but it is generic enough to be used by any cyber security researcher.
