Implementation of Matrix https://spec.matrix.org/

Spec

  • Server-Server API: https://spec.matrix.org/v1.3/server-server-api/
    • Client-Server API: https://spec.matrix.org/v1.3/client-server-api/
      • Versions: https://spec.matrix.org/v1.3/client-server-api/#get_matrixclientversions
  • Application Service API: https://spec.matrix.org/v1.3/application-service-api/
  • Identity Service API: https://spec.matrix.org/v1.3/identity-service-api/
  • Push Gateway API: https://spec.matrix.org/v1.3/push-gateway-api/

Requirements

  • Rust: https://www.rust-lang.org/tools/install
  • Flutter: https://docs.flutter.dev/get-started/install

Components

  • Dart matrix http API client: /crates/client
  • Core functionality: /crates/core
  • Cryptography: /crates/crypto
  • Rust matrix lib: /crates/lib
  • Matrix homeserver: /crates/server
  • Interfaces for application Services: /crates/services
  • Flutter based cross platform app (matrix client): /crates/services

Start Matrix homeserver

# The .toml file is used for the server configs, create youre own 
# this file is required to run the server.
$ cargo run -- ./crates/server/example_server.toml

Implemented APIs

Client-Server

  • versions: GET /_matrix/client/versions
