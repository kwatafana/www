      1 # Sqlite Support
      2 
      3 Sqlite is supported in cyrtophora as an optional feature:
      4 
      5 ```toml
      6 cyrtophora = { path = "../../cyrtophora/phora", features = ["sqlite"] }
      7 ```
      8 When the sqlite feature is enabled user account data will be saved in
      9 a sqlite database.
