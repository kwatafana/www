     25 ## HTOML file example (contents of a .toml)
     26 
     27 ``` toml
     28 # Title of the HTML page
     29 title = "Homepage"
     30 
     31 # Path to html output
     32 output = "publish/index.html"
     33 
     34 # Different sections for the HTML document
     35 [sections]
     36 
     37 # Navigation section [Optional]
     38 [sections.navbar]
     39 # Variant to use
     40 variant = "basic"
     41 # Links for the navigation
     42 links = [{url= "/", text= "Home"}, {url= "/about", text= "About"}, {url= "/contact", text= "Contact Us"}]
     43 
     44 # Hero section [Optional]
     45 [sections.hero]
     46 # Variant to use
     47 variant = "basic"
     48 heading = "Awesome Hero"
     49 sub_heading = "This is a cool sub heading"
     50 ```
     51 
     52 ## Usage
     53 
     54 To generate .html files from .toml files run the following command:
     55 
     56 ``` bash
     57 # This an example, use your own .toml files
     58 $ htoml index.toml about.toml contact.toml
     59 ```
