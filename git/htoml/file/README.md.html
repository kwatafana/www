     16 ## HTOML file example (contents of a .toml)
     17 
     18 ``` toml
     19 # Title of the HTML page
     20 title = "Homepage"
     21 
     22 # Path to html output
     23 output = "publish/index.html"
     24 
     25 # Different sections for the HTML document
     26 [sections]
     27 
     28 # Navigation section [Optional]
     29 [sections.navbar]
     30 # Variant to use
     31 variant = "basic"
     32 # Links for the navigation
     33 links = [{url= "/", text= "Home"}, {url= "/about", text= "About"}, {url= "/contact", text= "Contact Us"}]
     34 
     35 # Hero section [Optional]
     36 [sections.hero]
     37 # Variant to use
     38 variant = "basic"
     39 heading = "Awesome Hero"
     40 sub_heading = "This is a cool sub heading"
     41 ```
     42 
     43 ## Usage
     44 
     45 To generate .html files from .toml files run the following command:
     46 
     47 ``` bash
     48 # This an example, use your own .toml files
     49 $ htoml index.toml about.toml contact.toml
     50 ```
