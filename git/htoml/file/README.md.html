     23 ## HTOML file example (contents of a .toml)
     24 
     25 ``` toml
     26 # Title of the HTML page
     27 title = "Homepage"
     28 
     29 # Path to html output
     30 output = "publish/index.html"
     31 
     32 # Different sections for the HTML document
     33 [sections]
     34 
     35 # Navigation section [Optional]
     36 [sections.navbar]
     37 # Variant to use
     38 variant = "basic"
     39 # Links for the navigation
     40 links = [{url= "/", text= "Home"}, {url= "/about", text= "About"}, {url= "/contact", text= "Contact Us"}]
     41 
     42 # Hero section [Optional]
     43 [sections.hero]
     44 # Variant to use
     45 variant = "basic"
     46 heading = "Awesome Hero"
     47 sub_heading = "This is a cool sub heading"
     48 ```
     49 
     50 ## Usage
     51 
     52 To generate .html files from .toml files run the following command:
     53 
     54 ``` bash
     55 # This an example, use your own .toml files
     56 $ htoml index.toml about.toml contact.toml
     57 ```
